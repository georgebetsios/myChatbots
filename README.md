# 🤖 Conversational AI Chatbots (Dual-Framework Repository)

This repository contains two different implementations of conversational AI chatbots, exploring both modern Machine Learning/NLU approaches and traditional rule-based pattern matching.

---

## 📂 Subprojects Overview

This repository is split into two main sections:

### 1. 🤖 [Rasa Chatbot](./rasa-chatbot)
An advanced, NLU-driven conversational agent powered by the **Rasa Open Source framework**. It handles intent classification, entity extraction, and machine-learning-based dialogue management.
- **Technologies:** Python, Rasa NLU, Rasa Core, YAML

### 2. 💬 [AIML Chatbot](./aiml-chatbot)
A lightweight, rule-based chatbot implementation built using **AIML** (Artificial Intelligence Markup Language). It uses pattern matching and structured rules to process and respond to user prompts.
- **Technologies:** Python, AIML, XML

---

## 🛠️ Tech Stack & Concepts

- **Frameworks & Languages:** Python, Rasa, AIML, YAML, XML
- **Core Concepts:** Natural Language Understanding (NLU), Natural Language Processing (NLP), Intent & Entity Recognition, Rule-Based AI
